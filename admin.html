How This Works

Client-Side Tracking: All data is stored locally in your visitors' browsers using localStorage.

Privacy-Friendly: No external services, no cookies, no personal data collection.

Unique Visitors: Estimated using browser fingerprinting (screen size, timezone, user agent).

Sessions: New session after 30 minutes of inactivity.

Data Retention: Daily visit data is kept for 30 days automatically.

Limitations: Data resets if visitors clear their browser data. Only tracks visitors with JavaScript enabled.
